# Checklist de Producción

## WhatsApp

- Número dedicado confirmado.
- WhatsApp Cloud API configurado.
- Phone Number ID validado.
- Webhook verificado.
- Plantillas necesarias aprobadas.
- Opt-out probado.

## n8n

- Workflows importados.
- Credenciales configuradas en n8n.
- Webhook HTTPS activo.
- Error monitor activo.
- Export JSON guardado en el repo.

## Assistant

- Prompt aprobado.
- Servicios aprobados.
- FAQs aprobadas.
- Reglas de agenda aprobadas.
- Reglas de handoff aprobadas.
- Casos sensibles probados.

## Integrations

- Google Sheets probado.
- Google Calendar probado.
- Telegram o email probado.
- CRM externo probado si aplica.

## Testing

- Saludo.
- Pregunta de servicio.
- Pregunta de precio.
- Solicitud de cita.
- Reprogramación.
- Cancelación.
- Handoff humano.
- Opt-out.
- Payload malformado.
- Fallo de API.

## Go Live

- Cliente aprueba tono y respuestas.
- Equipo interno sabe cómo intervenir.
- Backup de workflows creado.
- Monitoreo inicial definido.
